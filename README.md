# Prepmate

This is a retro terminal UI app for generating quizzes, flashcards, and notes.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GROQ_API_KEY` in `.env.local` to your Groq API key
3. Run the app:
   `npm run dev`
